# 旅行手札照片檢視器

這是一個免安裝的本機照片檢視小工具。把照片放在任意資料夾中，打開 `index.html`，選擇該資料夾即可快速瀏覽，並以旅行手札的方式翻閱照片。

## 使用方式

1. 用瀏覽器打開 `index.html`。
2. 點「選擇本機照片資料夾」。
3. 選取放照片的資料夾。
4. 預設會進入一般瀏覽模式，點左側清單可切換右側照片。
5. 需要相簿感時，點「旅行手札模式」再點封面開始翻閱。
6. 手札模式可使用播放功能，並可調整播放速度。

## 功能

- 支援 JPG、PNG、GIF、WebP、BMP、AVIF。
- 照片僅供本機瀏覽，不會儲存或上傳任何檔案。
- 可依檔名或日期排序。
- 預設一般瀏覽，不顯示手札封面。
- 可切換旅行手札模式，顯示封面、章節與播放控制。
- 左側清單會依照片日期分成章節，並顯示獨立小圖預覽。
- 手札模式可自動播放照片並調整速度。
- 可切換「適合視窗」與「原始大小」。
- 可點縮圖或「放大觀看」開啟大圖模式。
- 大圖右下角會顯示拍攝日期。
- 可進入全螢幕檢視。

## 照片資訊

拍攝日期會從 JPG 照片的 EXIF 讀取，只顯示西元年月日，不顯示時分秒。若照片沒有 EXIF 日期，會使用檔案日期。

## 本機預覽伺服器

如果瀏覽器不允許直接開啟本機檔案，可以在這個資料夾執行：

```sh
python3 -m http.server 8765
```

再開啟：

```text
http://localhost:8765/index.html
```
